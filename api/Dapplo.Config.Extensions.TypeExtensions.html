Default(Type)

Create a default value for a type, this usually is "null" for reference type, but for other, e.g. bool it's false or for int it's 0 This extension method takes care of this.

Note: this differs a LOT from CreateInstance, as there we get an instance (e.g. for IList of string an empty List of string).
