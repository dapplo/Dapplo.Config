Class ConfigurationBase<TProperty>

An abstract non generic ConfigurationBase. This defines the API for the configuration based implementations. If you want to extend the functionality, extend this (or other classes) and implement a void xxxxxxxxGetter(GetInfo) or void xxxxxxxxSetter(SetInfo) which has a InterceptOrderAttribute
