ReloadAsync(Boolean, CancellationToken)

This is reloading all the .ini files, and will refill the sections. If reset = true, ALL setting are lost Otherwise only the properties in the files will overwrite your settings. Usually this should not directly be called, unless you know that the file was changed by an external process.
